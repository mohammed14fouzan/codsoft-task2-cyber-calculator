🧮CYBER CALCULATOR

CodSoft Web Development Internship-Task2

📝Project Description:

*This is a fully functional, interactive Web Calculator developed as part of my internship at CodSoft. The project focuses on creating a seamless user interface that mimics a real-world physical calculator while ensuring precise mathematical logic.
*The application handles basic arithmetic operations, decimal calculations, and clear-screen functions, providing a responsive experience across desktops, tablets, and smartphones.

✨Project Features:

*Basic Arithmetic: Support for Addition (+), Subtraction (-), Multiplication (×), and Division (÷).

*Floating Point Logic: Capability to handle decimal calculations accurately.

*Clear & Delete: Integrated 'AC' (All Clear) and 'DEL' (Delete last digit) functionality.

*Responsive Design: A mobile-first approach ensuring the UI looks great on any screen size.

*Glassmorphism UI: A modern aesthetic featuring frosted glass effects and subtle shadows.

*Keyboard Support: (Optional/If added) Input can be provided via mouse clicks or numeric keyboard keys.

🛠Tech Stacks:

*HTML

*CSS

*JS
